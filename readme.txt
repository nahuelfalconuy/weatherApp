# 🌦️ WeatherApp

## 📝 Descripción

Aplicación web que permite consultar el clima actual de una ciudad utilizando la API de [OpenWeatherMap](https://openweathermap.org/api). El usuario puede ingresar el nombre de una ciudad y obtener información como temperatura, humedad y condiciones generales del clima.

## 🔧 Tecnologías utilizadas

- **JavaScript** (vanilla)
- **HTML5**
- **CSS3**
- **Fetch API**
- **API REST (OpenWeatherMap)**
